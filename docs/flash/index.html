MimiClaw Flasher

Flash your ESP32-S3 device straight from this page. Chrome or Edge on desktop, USB cable, no software install.

1
Plug device into computer via USB.
2
Click Connect & Flash below, then pick the device's serial port when the browser prompts.
3
Wait for flashing to finish. Device reboots automatically.
4
On your phone, join the MimiClaw-XXXX WiFi hotspot it creates, then open 192.168.4.1 to enter WiFi + API key.
Your browser doesn't support Web Serial. Open this page in desktop Chrome or Edge.
Web Serial needs HTTPS. You're seeing this over an insecure connection.

This erases the device and writes the latest MimiClaw release.

Prefer the command line? See the latest release for esptool.py instructions.
